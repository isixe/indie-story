App Store launch is a rite of passage for every mobile developer. Here's the complete walkthrough.

## Preparation

### 1. Developer Account
- Individual: $99/year
- Company: $99/year, requires DUNS number
- Enterprise: $299/year, internal distribution

### 2. App Metadata
- App name (30 chars max)
- Subtitle (30 chars max)
- Keywords (100 chars max)
- Description (4000 chars max)
- Screenshots (5, specific sizes)
- Preview video (optional)

## Submission Process

### 1. Create App ID
Create a unique App ID in Apple Developer Portal.

### 2. Certificates & Profiles
Generate distribution certificate and App Store provisioning profile.

### 3. Build & Upload
Archive via Xcode, upload through Transporter or Xcode.

### 4. Fill App Information
Complete all required fields in App Store Connect.

### 5. Submit for Review
Typically takes 1-3 business days.

## Review Pitfalls

Common rejection reasons:
- Crashes or obvious bugs
- Missing privacy policy
- Test content left in
- Misleading description
- Hot update code

## ASO Optimization

- Keyword research: App Annie, Sensor Tower
- Title optimization: include core keywords
- Screenshot design: highlight key features
- Rating management: respond to reviews promptly

Launch is just the beginning — continuous optimization is where the real work starts.
