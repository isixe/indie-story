## 前言

大家好，我是一名独立开发者，如果还要详细一点的话，那么我是一名做了三年的网页端应用的独立开发者！

很多人谈到独立开发者，首先想到的就是赚钱吗？能养活自己吗？收入来源的方式是什么？...

总之，谈到独立开发者，归根节底离不开"钱"这个字。

那么，接下来我就谈谈我做独立开发者这些年的心路历程，可以算作是对这些年做独立开发者的一个总结吧！当然也可以给目前想做独立开发者或者正在做独立开发者的小伙伴一些经验吧。

**独立开发网站：**[91化简](https://91huajian.cn/)

## 1. 萌芽期

我大学学的是计算机相关专业，所谓相关专业，就是和计算机沾一点边，但是又不完全是学计算机的。这就导致了整个大学期间自己的编程能力是很薄弱的。

大概是从大三开始，每天在网上看到这种各样的软件、应用、网页等等，就很想开发一款属于自己的应用，当时不为别的，单纯的就是想学习技术，并且将自己天马行空的想法能够具象化出来。

可是由于自己的技术能力不足，加上那个时候的惰性很强，每当一开始没多久就放弃了。

后来就先选择了一条比较简单易上手的方式：**开通微信公众号。**

当时已经开始学习前端了，于是公众号最开始就分享各种前端知识，虽然坚持了一段时间，但是阅读量一直寥寥无几，很快就搁置了。

直到毕业，也没有上手开发自己的应用，一直都在断断续续的学习前端技能，唯一断断续续坚持的就只有微信公众号了。

## 2. 搭建个人博客（毕业第一年）

毕业之后，虽然前端的技术能力算不上多强，但是也找到了一份前端开发的相关工作，于是便开始了自己的程序员之路。

刚开始工作那一年，便全身心的投入了工作当中去，虽然心中一直还有着"开发一款属于自己的应用"的想法，但是被长时间耽搁了。

后来工作没有那么忙之后，就开始着手于搭建个人博客了，因为当时经常看到这样一句话：**一个好的程序员，应该有自己的独立博客。**

当时我也想有一个属于自己的一片天地，能够记录自己的生活以及技术积累。于是便开始搭建个人博客了，本着免费的原则，当时直接利用 GitHub 搭建了一款属于自己的免费个人博客。

在后来，我还专门写了一篇文章，如何利用 GitHub 免费搭建个人博客，不过也是好几年前了。

![](/content/indie-dev-self-narrative/blog-github-pages.png)

博客开通了一段时间，觉得访问速度太慢了，于是便自己买了一台服务器和域名，准备自己搭建一个。

当时域名备案搞了挺长一段时间的，因为国内不允许域名备案选择个人博客一类的。

最终用了一个开源的博客框架，终于将博客部署到了自己的服务器上面去。接下来就开始了写文章之旅，并且我还将文章同步发表到了公众号上面去。

但是没过几个月，我还是放弃了，当时主要觉得是以下几点影响的吧：

1.  **服务器到期了，续费太贵了。**
2.  **自己想法太多，老是想更改博客的各种功能和样式，但是利用框架搭建的博客，可塑造性又不强。**
3.  **写文章和维护独立博客过于麻烦，当时想专注于某一项。**
4.  **没人看文章，没有动力。**

## 3. 全力做公众号（毕业第二年）

搭建个人博客这条路放弃之后，我便开始思考新的方向了。最终落在了做公众号这一方向上，因为到目前为止，唯一还在正常维护的也就只有那个公众号了。

但是之前那个公众号主要分享前端知识、文章等等，受众面太窄了。当时已经工作两年了，我也不再是当初那个单纯的我了，当时只想着挣钱，所以想着能不能将公众号发展成副业。

于是我重新更名了公众号，并且换了分享内容的方向，主要分享各类网站、黑科技工具、软件等等。

那一年我每天下班回家之后的第一件事就是开始准备公众号文章，然后设置定时发送，第二天一大早发送出去。

经过一年的努力，公众号粉丝逐渐变多了，从最开始的 10 多个粉丝、到 100 个、到开通流量主，然后 1000 粉、5000 粉，到现在的 1 万粉丝。

并且我在更新公众号的时候，还同步开发了小程序。当时小程序的用户也有 1 万用户左右了。

但是小程序对于个人开发者确实太不友好了，大部分功能辛辛苦苦开发出来，没两天就给你永久封禁了，没办法，只能放弃个人小程序了。

![](/content/indie-dev-self-narrative/wechat-platform-income.png)

那一年基本上没有停歇过，每天都会去自己的公众号平台看看，更新文章。

后来粉丝量达标之后，流量主会有一点收入，然后也可以到一些平台接一点广告。

但是总体算下来收入真的很低，到目前为止，流量主的收入总共也就一千多块钱，加上接的一些广告，大约也就两三千块钱吧。

总结而言，就是真的没有网上那些人吹嘘的什么月入过万，睡后收入等等。

![](/content/indie-dev-self-narrative/traffic-master-income.png)

努力了一年多，收入很少，所以我也打算先暂停一下这方面的投入，因为确实投入太多，但是效果不明显了。

## 4. 开始写文章、录视频（毕业第三年）

公众号从第三年开始就是断断续续去更新了，没有投入太大的精力到上面去了。

但是我就是闲不住，自己的抱负始终没有实现，总感觉自己的梦想还在，还有机会，于是又开始了新的方向。

工作了三年了，自己的前端技能也有所长进，所以打算开始写一些前端方面的文章，以及录制相关视频。

但是这一次我没有再搭建个人博客，这次我打算直接将文章投稿各个平台，开始专注于写技术文章。

于是，我便开始在三个平台疯狂输出：[知乎](https://www.zhihu.com/people/luhongquan)、[掘金](https://juejin.cn/user/3034307822112798)、[哔哩哔哩](https://space.bilibili.com/493520625?spm_id_from=333.1007.0.0)

当时我能连续更文一个月，中间不间断，并且每一篇文章都是下班回家后从零开始编写。

现在回想起那个时候，真的挺有激情的，每天晚上写完文章之后，还继续为这篇文章录制一个讲解视频。

就这样断断续续坚持了一年左右，大概写了一两百篇文章吧。

虽然那一年并没有产生什么收入，但是却感觉很充实，也收获了一些粉丝。

![](/content/indie-dev-self-narrative/zhihu-profile.png)
![](/content/indie-dev-self-narrative/juejin-profile.png)
![](/content/indie-dev-self-narrative/bilibili-profile.png)

当时给我很大激励的当属掘金了，每次参加更文活动会收获不少礼品，当时非常开心。

![](/content/indie-dev-self-narrative/juejin-prizes.png)

努力了一年，收获还是很多的：收获了粉丝、收获了知识、也锻炼了自己的意志。

但是终究和我心中所想有所差距，我想要的是一款自己的应用、一个对自己有意义的副业。

## 5. 开发网址导航网站（毕业第三年结束）

通过前几年的洗礼，感觉自己的技术能力得到了很大的提升，自己已经有能力独立开发一套完整的应用了。

于是便开始思考要做什么了，当然主要集中在 web 网页端，因为自己一直还是从事的网页端开发工作。

最终锁定在了一个比较简单易上手的项目：网址导航项目。

为什么选择这个项目，当时主要考虑到下面几个点：

1.  网址导航项目容易开发，功能较简单
2.  我的个人公众号与此类项目有共通之处
3.  我个人确实有收集网站的需求

确定好了项目于是就开始着手开发了，一边着手开发，一边购买服务器和域名注册以及备案。

项目的开发周期比较短，但是比较痛苦的。

那时候上班一闲下来，我就会开发我的网站导航网站，下班回家后也继续开发网站。

总之那一两个月，我基本上也没怎么休息，其主要原因还在于开发过程中会遇到很多问题、以及很多新技术。都需要我一一去攻克以及学习。

经过几个月的开发以及各项资料的准备，网站总算是上线了，当时网站名称就叫：小猪导航。

直到前两个月，我目前的 91 化简产品都还在使用 smllpig.site 这个域名作为文件服务器，到现在为止，这个域名已经彻底被我注销了，因为没有正在运行的网站了，备案无法通过，所以只能注销了。

当时小猪导航上线之后，每天也还是有好几百的访问量，并且百度也收录了，但是最终我还是放弃了这个项目，主要有下面几个原因吧：

1.  本着练手的态度开始的这个项目，主要是为了摸清楚技术以及开发思路。
2.  网址导航项目比较难商业化，广告也比较难接。
3.  个人有了新的想法。

## 6. 91化简

[91化简](https://91huajian.cn/) 这个项目是我目前正在运营的项目，也是从 22 年底到目前为止一直在维护的一个项目。

22 年底的时候网站导航项目基本上也稳定了，于是便开始这个新的项目。

这个项目相较于之前的项目，都要复杂一点，因为我想要的是一个比较完整的项目、并且技术栈也是比较新的一个项目。

这里我简单介绍一下我这个项目：

> 91化简是一个开源免费的简历设计制作以及提供模板下载的网站，前端项目完全开源免费。
> 网站提供有完整的后台管理系统，可以方便管理整个网站。
> 网站内置两款设计器、快速设计、简历、封面、海报均可免费制作，支持一键导出高清 PDF、JSON 数据等。

网站的前端工程一直都是免费开源的，也得到了大家的喜爱，目前 GitHub 上的 star 数已经超过了 1.5k。

这个项目也是目前为止我最喜欢的一个项目，我付出了很多心血到这个项目上去，也是通过这个项目，让我觉得我是一个独立开发者。

## 7. 独立开发者之路

回想过去的五年，彷佛自己没有休息过，并不是我不想休息，也不是我多么的爱工作。

反之，我无时无刻都想全身心的放松下来，好好的休息一番。

但是我做不到，只是因为我心中的梦想始终没有实现。

有人会问我，五年时间，做了这么多事，收入了多少？

我依托事实来回答：收入真的不多，这几年的回报还远远达不到个人的付出。

但是我不后悔，因为我的方向逐渐明朗了起来，人这一生能找到一件自己喜欢的事不容易。

到目前为止，我可以自豪的称自己为一名独立开发者了，并不是因为我通过我的应用赚了多少钱，而是我通过我的项目开始觉得生活有了更多的意义。

在网上我经常看到很多关于独立开发者的文章或者帖子，常常有人会说一大堆，比如会告诉你开发什么应用、踏入某个领域、某种软件做不得等等之类的。

但是我只想说，如果想要成为独立开发者，必须要做到下面两个字：

**坚持！**

我也会坚持下去的：

-   坚持将项目不断更新维护
-   坚持学习网站运营知识
-   坚持初心
-   坚持这份韧劲
